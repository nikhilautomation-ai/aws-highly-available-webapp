AWS Cloud Resume Project

Highly Available Web Application on AWS

This application demonstrates EC2, Auto Scaling, Application Load Balancer, S3, IAM, CloudWatch, Security Groups, and Nginx.

High Availability

EC2 instances are managed through an Auto Scaling Group across multiple Availability Zones.

Artifact Management

Application revisions are stored in Amazon S3 and served from EC2 running Nginx.

Monitoring

CloudWatch monitors infrastructure metrics, Nginx logs, and alarms.

Security

IAM roles and security groups provide controlled service access.

Application Version: 2.0

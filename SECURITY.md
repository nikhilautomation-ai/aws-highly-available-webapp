# Security Policy

This public repository intentionally excludes:

- AWS account IDs and IAM ARNs
- EC2 instance IDs and public IP addresses
- VPC, subnet, security group, target group, and launch template IDs
- S3 bucket names used during testing
- AWS console screenshots
- PEM files, access keys, secret keys, tokens, passwords, and `.env` files

The infrastructure used for the project was deleted after testing to avoid ongoing charges.

Please do not commit credentials or private AWS resource identifiers to this repository.
